Reviewable deliverables an agent produces for an issue (UX shots, mockups, concept docs, HTML/MD proposals) must be attached to the issue via podium issue artifact --add, from a durable repo/worktree path (scratchpad and /tmp paths do not render). Publishing elsewhere (chat upload, external artifact link) does not replace attaching it. Decision: Tillmann, 2026-07-20 (POD-98).
