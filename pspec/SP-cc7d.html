Podium must NOT run as root. On the dev/prod box it runs as a dedicated podium user (decided 2026-07-12): superagent turns execute headlessly with bypassPermissions, so the OS user is the only blast-radius boundary — root would hand every autonomous turn the whole machine.

Running as root in containers remains supported: the daemon sets IS_SANDBOX=1 for headless claude turns when uid is 0 (see apps/daemon/src/headless-drivers.ts).
